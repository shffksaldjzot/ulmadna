## 2. 서비스 핵심 구조

### 2-1. UI 패턴: 원페이지 실시간 계산기

**스텝퍼(Wizard) UI 사용 금지.** 숨고, 토스, 오늘의집처럼 단계별 화면 전환 방식을 쓰지 않습니다.

모든 입력과 결과가 **한 화면**에 보이고, 옵션을 바꿀 때마다 결과가 **즉시 업데이트**됩니다.

### 2-2. 3단 Progressive Disclosure

| 단계 | 대상 | 소요 | 보이는 것 |
|---|---|---|---|
| 1단계 | 90% 사용자 | 3초 | 평형+유형+등급만 선택 → 총액 |
| 2단계 | 30% 사용자 | 30초 | 공정 탭 열면 핵심 2~4개 칩 |
| 3단계 | 5% 사용자 | 3분 | "상세설정▸" 누르면 브랜드/모델/세부옵션 |

### 2-3. 레이아웃

**데스크톱:** 좌우 분할 — 입력 55~60% : 결과 40~45%. 결과 패널은 sticky.
**모바일:** 세로 배치 — 총 금액이 항상 상단에 보임.

상세 레이아웃은 `얼마드나_디자인_브리프.md`를 참조하세요.

- 구축10년 → 밑작업 "퍼티"
- 구축20년+ → 밑작업 "퍼티+초배"

## 6. 결과 표시

### 6-1. 공정별 금액 리스트 (실시간 업데이트)
```
철거          230만원
설비           120만원
방수            30만원
창호         1,150만원
...
──────────────
공사비 소계    4,357만원
예비비(10%)     436만원
──────────────
예상 총 비용   4,793만원
평당 145만원
```

### 6-2. 숨은 비용 알림 (항상 표시)
에어컨/샷시 등 추가비용을 **개별 분리**해서 보여줍니다:
```
⚠️ 별도 발생 가능한 추가 비용:
  [에어컨] 보양비 60만 / 전기공사 30만 / 앵글 20만 / 스킬도배 30만
  [샷시] 실측비 / 양중비 / 철거비 / 통바 약 132만
  [기타] 사다리차 13~40만 / 엘리베이터 사용료 10~44만
```

### 6-3. 다운로드 버튼 4종
```
[PDF] [엑셀] [이미지(PNG)] [링크복사]
```
- PDF: 2페이지 — 1페이지 견적표 + 2페이지 체크리스트 + 배너 광고
- 엑셀: 동일 데이터 스프레드시트 형태
- 이미지: 카톡 공유용 간소화 버전 (600px 폭)
- 링크복사: 현재 설정값이 URL 파라미터로 저장

견적서 디자인은 `얼마드나_견적서_디자인.html`과 `얼마드나_견적서_2페이지_체크리스트.html`을 참조하세요.
