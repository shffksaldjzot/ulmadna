# PART 0. 집 보러 갔을 때 체크리스트

> 매매·전세·월세 — 집을 처음 보러 갔을 때 인테리어 관점에서 확인할 항목
> 이 체크리스트를 기반으로 인테리어 비용을 사전 추정할 수 있습니다

---

## A. 전체 컨디션 파악

- [ ] 준공연도 확인 (등기부등본 또는 관리사무소) — 구축 연수 파악
- [ ] 최근 인테리어 이력 확인 (직전 소유자/세입자가 했는지)
- [ ] 전체 면적 확인 (공급면적 vs 전용면적)
- [ ] 확장 여부 확인 (발코니 확장 유무)
- [ ] 구조 변경 여부 확인 (벽 철거·가벽 추가 이력)
- [ ] 향·층수·동 확인 (결로·곰팡이·채광 관련)
- [ ] 전체적인 냄새 확인 (곰팡이·담배·음식·하수구 냄새)

## B. 바닥 상태

- [ ] 바닥재 종류 확인 (장판/마루/타일/원목)
- [ ] 바닥 수평 확인 (구슬 또는 물병 테스트)
- [ ] 바닥 들뜸·부풀음 확인 (밟아보면서 확인)
- [ ] 바닥 스크래치·찍힘 정도
- [ ] 이음매 벌어짐 확인
- [ ] 걸레받이 상태 확인 (들뜸·탈락)
- [ ] 난방 배관 상태 추정 (바닥 특정 부분만 뜨거운지 — 배관 누수 의심)
- [ ] 문턱(문지방) 높이 확인 — 바닥재 교체 시 레벨 참고

## C. 벽면·천장 상태

- [ ] 벽지 종류 확인 (합지/실크/페인트)
- [ ] 벽지 들뜸·변색·오염 정도
- [ ] 벽면 곰팡이 확인 (**특히 외벽, 욕실 인접 벽, 베란다 접합부**)
- [ ] 벽면 균열·크랙 확인 (구조적 크랙 vs 표면 크랙 구분)
- [ ] 천장 상태 확인 (누수 자국·얼룩·처짐)
- [ ] 천장 높이 확인 (우물천장 시공 가능 여부 — 최소 2.3m 이상)
- [ ] 몰딩 상태 확인 (탈락·변색)
- [ ] 결로 흔적 확인 (창틀 주변 검은 곰팡이)

## D. 창호/샷시 상태

- [ ] 샷시 브랜드·연식 확인 (20년 이상이면 교체 권장)
- [ ] 모든 창문 열고 닫아보기 (뻑뻑함·소음)
- [ ] 잠금장치 작동 확인
- [ ] **샷시 세게 닫아보기 — 흔들림 확인**
- [ ] 창틀 주변 실리콘 상태 (갈라짐·곰팡이)
- [ ] 유리 종류 확인 (단창/이중창/로이)
- [ ] 외풍 확인 (손을 대보거나 라이터 불 흔들림으로 체크)
- [ ] 방충망 상태
- [ ] 결로 심한지 확인 (겨울철 방문 시 필수)

## E. 욕실 상태

- [ ] 욕실 개수 확인
- [ ] 타일 상태 (균열·들뜸·줄눈 변색)
- [ ] **타일 두드려보기 — 공극(들뜸) 확인** (텅텅 소리 나면 들뜬 것)
- [ ] 양변기 상태 (급수·배수 작동 — 물 내려보기)
- [ ] 세면기 배수 확인 (물 틀어보기)
- [ ] 샤워 수전 온수/냉수 작동 확인
- [ ] 수압 확인 (세면기+샤워 동시 사용)
- [ ] 배수 속도 확인 (바닥 배수구에 물 부어보기)
- [ ] 환풍기 작동 확인
- [ ] 천장재 상태 (SMC 변색·처짐)
- [ ] **바닥 기울기 확인 — 물이 배수구 쪽으로 흐르는지** (물 부어보기)
- [ ] 방수 상태 추정 (벽·바닥 경계 곰팡이·물때)
- [ ] 실리콘 상태 (변색·곰팡이·갈라짐)
- [ ] 악취 확인 (배수 트랩 문제)

## F. 주방 상태

- [ ] 싱크대 연식·상태 확인
- [ ] 상판 스크래치·변색·갈라짐
- [ ] 하부장 문짝 개폐 상태 (경첩 헐거움)
- [ ] 서랍 레일 작동 상태
- [ ] 싱크볼 배수 확인 (물 틀어보기)
- [ ] 수전 작동 확인 (온수/냉수)
- [ ] 후드 작동 확인 (흡입력)
- [ ] 가스레인지/인덕션 작동 확인
- [ ] 상부장 고정 상태 (흔들림)
- [ ] 냉장고 자리 사이즈 확인 (뚱냉 대응 가능한지)
- [ ] 주방 벽타일 상태

## G. 전기·설비 상태

- [ ] 분전반 확인 (차단기 라벨링·용량)
- [ ] 분전반 연식 확인 (20년 이상이면 교체 권장)
- [ ] 콘센트 개수·위치 확인 (부족하면 추가 필요)
- [ ] 콘센트 통전 확인 (충전기 꽂아보기)
- [ ] 스위치 작동 확인
- [ ] 인터폰 작동 확인
- [ ] 도어락 작동 확인
- [ ] 조명 상태 확인 (종류·밝기)
- [ ] 보일러 연식·상태 확인 (15년 이상이면 교체 고려)
- [ ] 보일러 에러 코드 확인
- [ ] 난방 온도조절기 작동 확인
- [ ] 에어컨 유무·상태 확인 (배관 매립 여부)

## H. 현관·복도·베란다

- [ ] 현관문 상태 (개폐·잠금·도어클로저)
- [ ] 현관문 방화등급 확인
- [ ] 현관 바닥 상태 (타일 들뜸·균열)
- [ ] 신발장 상태
- [ ] 복도 폭 확인 (가구 반입 가능 여부)
- [ ] 베란다 바닥 상태 (균열·방수 상태)
- [ ] 베란다 천장 누수 흔적 확인
- [ ] 베란다 배수구 확인 (막힘 여부)
- [ ] 세탁기 자리·배수 확인
- [ ] 건조기 자리·환기구 확인

## I. 공용부 확인

- [ ] 엘리베이터 상태·크기 (대형 가구·자재 반입 가능 여부)
- [ ] 계단 상태 (비상시 + 자재 운반)
- [ ] 복도 폭 (자재 양중 동선)
- [ ] 주차장 (시공 차량 주차 가능 여부)
- [ ] 사다리차 접근 가능 여부 (건물 구조·나무 등 장애물)
- [ ] 관리사무소 공사 규정 확인 (시간·보양·엘베 사용료)
- [ ] 같은 동 시공 사례 확인 (관리사무소에 문의)

## J. 사진 촬영 (필수)

- [ ] 전 공간 전경 사진 (각 방·거실·주방·욕실·베란다)
- [ ] 문제 부분 클로즈업 (곰팡이·균열·누수·들뜸)
- [ ] 분전반 사진
- [ ] 보일러 모델·연식 사진
- [ ] 창호 브랜드 라벨 사진
- [ ] 욕실 천장·배수구 사진

- [ ] 평면도 또는 도면 (부동산에서 수령)

## A. 업체 검증

- [ ] 사업자등록증 수령 (설립일 확인 — 신설 1년 미만 주의)
- [ ] 법인 등기부등본 확인 (대표자명·본점·자본금)
- [ ] 건설업 면허 보유 여부 (면허번호 직접 조회)
- [ ] 포트폴리오 실제 시공 여부 (동일 아파트 단지 시공사례 우선)
- [ ] 인터넷 리뷰 확인 (네이버·셀인카페·오늘의집)
- [ ] 명함·회사소개서·계약서 업체명 3곳 일치 여부
- [ ] 사무실 실존 여부 (주소 확인, 가능하면 방문)
- [ ] 카톡 채널·홈페이지 운영 상태 확인
- [ ] 이전 고객 A/S 후기 확인 (시공 후 1년 이상 된 후기)
- [ ] 인감증명서 수령

## B. 견적서 확인

- [ ] 공정별 항목·수량·단가 분리 여부 ("턴키 일괄 ○○만원"만 적힌 견적 위험)
- [ ] 자재 브랜드·모델명 명시 여부 (미명시 시 저가 대체 가능)
- [ ] "추가금 없음" 서면 확인 (말로만은 법적 효력 없음)
- [ ] VAT 포함/별도 명시 (10% 차이 — 5천만원이면 500만원)
- [ ] 철거비·폐기물 처리비 포함 여부 (빠지면 100~300만 추가)
- [ ] 보양비·양중비·운송비 포함 여부 (빠지면 100~200만 추가)
- [ ] 에어컨 추가비 확인 (보양60+전기30+앵글20+도배30 = 140만 별도)
- [ ] 샷시 부가시공비 확인 (실측+양중+철거+통바 약 130만 별도)
- [ ] 승강기 사용료 별도 여부 (관리사무소 규정 10~44만)
- [ ] 사다리차 비용 별도 여부
- [ ] 준공청소 포함 여부
- [ ] 입주청소·나노코팅 포함/별도 여부
- [ ] 견적서 유효기간 명시 여부

## C. 계약 조건

- [ ] 계약금 비율 (권장: 계약금10% · 중도금40% · 중도금40% · 잔금10%)
- [ ] 입금 계좌가 법인 계좌인가 (대표 개인계좌 = 위험)
- [ ] 공정표(공사 일정표) 사전 수령
- [ ] 도면(평면도·전개도·3D) 사전 수령
- [ ] 마감재 샘플 미팅 완료 (실물 확인)
- [ ] A/S 기간 계약서 명시 (최소 1년 무상)
- [ ] 하자보수 범위 및 절차 명시
- [ ] 계약 해지 조건 명시 (7일 이상 착공 미이행 시 해지 가능)
- [ ] 위약금 조항 쌍방 공정성 확인
- [ ] 공사 지연 시 지체상금 조항
- [ ] 계약서 2부 작성 (쌍방 보관)
- [ ] 플랫폼 통해 계약 시 업체명 일치 확인
- [ ] 세금계산서 발행 여부 확인
- [ ] 하도급 여부 확인 (시공을 다른 업체에 넘기는지)
- [ ] 현장 담당자(현장소장) 지정 여부

## D. 관리사무소 확인

- [ ] 공사 가능 시간 확인 (보통 평일 09~18시, 토요일 09~13시)
- [ ] 공사 신고서 제출 (관리사무소)
- [ ] 주민동의서 징수 (층간 세대 동의)
- [ ] 엘리베이터 보양 규정 확인
- [ ] 엘리베이터 사용료 확인
- [ ] 자재 반입 경로 확인 (사다리차 가능 여부)
- [ ] 주차 가능 여부 (시공 차량)
- [ ] 소음·분진 관련 민원 대응 방안
- [ ] 현관문 교체 시 관리사무소 승인 여부
- [ ] 확장공사 시 구청 허가 여부

## E. 필수 백업

- [ ] 카카오톡 채팅 내용 수시 백업 (팀채팅 강퇴 대비)

# PART 2. 공사 중 체크리스트

## A. 철거 단계

### 철거 전
- [ ] 보양 작업 완료 확인 (엘리베이터·복도·바닥·현관)
- [ ] 보존할 부분 표시 (테이핑·마킹)
- [ ] 전기·수도·가스 차단 확인
- [ ] 분전반 위치 확인·보호
- [ ] 욕실 배수구 막힘 방지 (비닐로 보호)
- [ ] 이웃 세대 공사 알림 (소음·분진)

### 철거 중
- [ ] 철거 범위가 견적서와 일치하는지 확인
- [ ] 구조벽(내력벽) 철거하지 않는지 확인 — 건물 구조 안전
- [ ] 바닥 난방 배관 손상 여부 확인
- [ ] 전기 배선 손상 여부 확인
- [ ] 폐기물 분리 반출 확인 (아스콘·콘크리트·목재·혼합)
- [ ] 석면 포함 자재 여부 확인 (2000년 이전 건물)

### 철거 후
- [ ] 바닥 레벨(수평) 확인
- [ ] 벽면 상태 확인 (곰팡이·균열·누수 흔적)
- [ ] 배관 상태 확인 (녹·누수)
- [ ] 현장 사진 촬영 (전 공간 360도)
