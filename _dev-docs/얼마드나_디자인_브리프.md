# 얼마드나 (ulmadna.com) — 디자인 브리프

---

## 서비스 소개

**서비스명:** 얼마드나
**도메인:** ulmadna.com
**한 줄 소개:** "우리 집 인테리어, 얼마 드나?" — 회원가입·전화번호 없이 바로 쓰는 무료 인테리어 예상 견적 계산기
**운영:** 도토리스튜디오 (인테리어 컨설팅 전문, 명품 브랜드 매장 시공 10년 경력)

---

## 서비스 콘셉트

기존 인테리어 견적 계산기(집닥, 오늘의집, 내공 등)는 전부 결과를 보려면 전화번호를 넣거나 상담 신청을 해야 합니다. 얼마드나는 이런 것 일절 없이, 들어오자마자 바로 계산하고 PDF/엑셀까지 다운로드할 수 있는 완전 무료 서비스입니다.

수익은 배너 광고와 자재 브랜드 스폰서십으로만 발생하며, 사용자에게는 어떤 비용도 요구하지 않습니다.

향후 부동산계산기(ezb.co.kr)처럼 견적 계산기 → 자재 소요량 계산기 → 견적서 비교 분석기 → 공사 기간 계산기 등으로 확장하여 "인테리어 계산은 얼마드나"라는 포지셔닝을 잡을 계획입니다.

## 브랜드 톤 & 무드

**키워드:** 고급 + 따뜻 + 전문적이지만 접근 가능한 + 투명한 + 부담 없는

도토리스튜디오의 기존 브랜딩과 연계하되, 얼마드나만의 가벼운 접근성을 추가합니다. 무겁거나 어두운 느낌이 아니라, "편하게 쓰세요"라는 느낌이 먼저 와야 합니다. 동시에 "이 숫자는 믿을 수 있다"는 전문성이 뒷받침되어야 합니다.

## 페이지 구조 & UI 패턴

### ❌ 사용하지 않는 패턴
스텝퍼(Stepper/Wizard) UI를 쓰지 않습니다. 숨고, 토스, 오늘의집처럼 한 단계씩 화면이 전환되는 방식은 이탈률이 높고, 광고 지면도 확보되지 않으며, "편하게 쓰세요" 콘셉트와 맞지 않습니다.

### ✅ 사용하는 패턴
원페이지 실시간 계산기. 입력과 결과가 한 화면에 보이고, 옵션을 바꿀 때마다 결과가 즉시 업데이트됩니다. 부동산계산기(ezb.co.kr)의 양도세 계산기 페이지와 유사한 구조입니다.

## 주요 UI 요소 설명

### 헤더
- 로고: "얼마드나" 텍스트 로고 (딥 브라운)
- 배지: "완전 무료 · 회원가입 없음 · 전화번호 없음" — 이 메시지가 헤더에서 항상 보여야 함
- 향후 확장 시 네비게이션 탭: [견적 계산기] [자재 소요량] [견적서 비교] ...

### 입력 패널 — 기본 조건
- 평형: 카드형 버튼 그룹 (10/20/24/30/34/40/50/직접입력)
- 주거 유형: 세그먼트 컨트롤 3개
- 지역: 드롭다운 또는 세그먼트
- 자재 등급: 3개 카드 (각 카드에 대표 자재 예시 + 브랜드명 노출)
  - 기본: "합지벽지 · 장판 1.8T · 국산 세라믹"
  - 중급: "실크벽지 · 강마루 6T · 포세린 타일 (LX Z:IN)"
  - 고급: "수입벽지 · 원목마루 · 수입 포세린"
- 천장 높이: 토글 또는 드롭다운 (기본값 2,300mm)

### 입력 패널 — 공정별 세부 선택 (내공 naegong.com 참고)

내공 스타일과 동일하게 공정별 섹션이 세로로 쭉 나열. 각 섹션 안에 세부 항목이 버튼(칩) 형태로 배치. 클릭하면 선택/해제. 개수는 드롭다운.

**내공 대비 개선점:**
- 각 버튼에 예상 금액 표시 (내공은 숫자 안 보임)
- 공정명 옆에 해당 공정 소계 표시
- 선택 변경 시 상단 총 금액 즉시 반영

### 결과 패널 — 총 예상 비용
- 가장 큰 숫자: "약 4,255만원" (Bold, 크게)
- 서브: "예비비 555만원 포함 · 평당 약 142만원"
- 합리성 배지: 초록(합리적) / 노랑(적정) / 빨강(다소 높음)
  - 예: 🟢 "시장 대비 적정 수준"

### 결과 패널 — 공정별 견적표
- 테이블: 공정명 / 금액 / 비중(%) / 수평 바
- 금액 큰 순서로 정렬
- 각 공정 옆에 작은 텍스트로 자재 기준 표시 (예: "강마루 6T 동화 기준")

### 결과 패널 — 비용 비중 차트
- 도넛 차트 또는 수평 바 차트
- 색상은 공정별로 구분

### 결과 패널 — 공정 순서 가이드
- 세로 타임라인 형태
- 각 공정명 + 예상 기간 (예: "철거 3~5일 → 설비 3~5일 → ...")

### 다운로드 / 공유 버튼
- PDF 다운로드 버튼 (골드 포인트 컬러)
- 엑셀 다운로드 버튼
- 카카오톡 공유 버튼
- 모두 회원가입 없이 즉시 동작

### 배너 광고 영역
- 결과 콘텐츠 아래, 도토리스튜디오 소개 위
- 가로형 배너 2~3개
- "광고" 라벨 표시
- 콘텐츠 영역과 시각적으로 구분되어야 함 (연한 배경색 등)

### 도토리스튜디오 소개
- 작고 겸손하게: "이 계산기는 인테리어 실무 경력 10년 전문가가 만들었습니다"
- 로고 + 한 줄 소개 + 링크

### 푸터
- 단가 출처: "2025~2026 시장 평균 기반 · 조달청 적산 교차 검증 예정"
- 면책: "본 견적은 참고용이며 실제 시공비는 현장 실측에 따라 달라질 수 있습니다"
- 도토리스튜디오 연락처

## 핵심 인터랙션

1. 입력 옵션 변경 시 결과가 실시간으로 업데이트 (별도 "계산하기" 버튼 없음)
2. 공정 토글 ON/OFF 시 총 금액이 즉시 반영
3. 모바일에서 입력 영역을 접으면 결과만 크게 보임
4. PDF/엑셀 다운로드는 클릭 즉시 생성 (로딩 인디케이터 표시)

## 회원 시스템 (선택적)

- 비회원: 모든 기능 100% 사용 가능
- 회원: "저장하기" 버튼 클릭 시에만 카카오 로그인 안내
- 가입 유도 팝업/모달 없음

---

## 참고 사이트

- 부동산계산기 (ezb.co.kr) — 원페이지 계산기 UI 패턴 참고
- 내공 (naegong.com) — 경쟁사, 우리가 이겨야 할 대상
- 집닥 견적 계산기 (zipdoc.co.kr/estimate) — 경쟁사
- 도토리스튜디오 — 모회사 브랜딩 톤 참고

---

## 디자인 산출물 요청

1. 데스크톱 메인 페이지 (입력+결과 좌우 분할)
2. 모바일 메인 페이지 (입력 접기 상태 + 결과 펼침 상태)
3. 결과 PDF 리포트 레이아웃 (A4)
4. 카카오톡 공유 카드 디자인
5. 로고 (텍스트 로고 "얼마드나" + 아이콘)

---

*얼마드나 디자인 브리프 — 도토리스튜디오*
