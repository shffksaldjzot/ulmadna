> 보충일: 2026.04.06
> 목적: 각 입력 필드의 타입(라디오/체크박스/수량), 비용 개별 분리, 3단 UI 구조 확정

---

## UI 3단 구조 요약

| 단계 | 대상 | 소요시간 | 보이는 것 |
|---|---|---|---|
| 1단계 | 90% 사용자 | 3초 | 평형+유형+등급 → 총액 |
| 2단계 | 30% 사용자 | 30초 | 공정별 핵심 2~4개 칩 |
| 3단계 | 5% 사용자 | 3분 | 브랜드/모델/세부옵션 전체 |

---

## 입력 타입 정의

- **Radio(●)**: 하나만 선택. 칩 토글 중 하나만 활성화.
- **Checkbox(☐)**: 다중 선택. 여러 개 동시 ON 가능.
- **Stepper(▾)**: 수량 선택. 드롭다운 또는 +/- 버튼.
- **Toggle(⊙)**: ON/OFF 스위치. 공정 전체 활성화/비활성화.
