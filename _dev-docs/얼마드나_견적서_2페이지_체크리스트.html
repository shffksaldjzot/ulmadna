1 업체 검증

사업자등록증 수령 완료 설립일 확인 — 신설 1년 미만 법인 주의
법인 등기부등본 확인 대표자명·본점 소재지·자본금 확인
건설업 면허 보유 여부 있다고 주장 시 면허번호 직접 조회
포트폴리오 실제 시공 여부 동일 아파트 단지 시공사례 우선 확인
인터넷 리뷰 확인 네이버·셀인카페·오늘의집 후기 검색
명함·회사소개서·계약서 업체명 일치 세 곳의 이름이 다르면 위험 신호

2 견적서 확인

공정별 항목·수량·단가가 분리되어 있는가 "턴키 일괄 ○○만원"만 적힌 견적은 위험
자재 브랜드·모델명이 명시되어 있는가 미명시 시 저가 자재로 대체될 수 있음
"추가금 없음"을 서면으로 확인했는가 말로만 약속하면 법적 효력 없음
VAT 포함/별도 여부 명시 10% 차이 — 5,000만원이면 500만원 차이
철거비·폐기물 처리비 포함 여부 빠져있으면 100~300만원 추가
보양비·양중비·운송비 포함 여부 별도인 경우 100~200만원 추가
에어컨 추가비 확인 보양60+전기30+앵글20+도배30=140만원 별도
샷시 부가시공비 확인 실측+양중+철거+통바 약 130만원 별도

3 계약 조건

계약금 비율 확인 계약금10% · 중도금40% · 중도금40% · 잔금10% 권장
입금 계좌가 법인 계좌인가 대표 개인계좌 입금 요구 시 위험
공정표(공사 일정표)를 받았는가 착공 전 서면 공정표 필수
도면(평면도·3D)을 받았는가 도면 없이 철거 시작하면 안 됨
마감재 샘플 미팅을 했는가 실물 확인 없이 시공하면 분쟁 발생
A/S 기간이 계약서에 명시되어 있는가 최소 1년 무상 A/S 권장
계약 해지 조건이 명시되어 있는가 7일 이상 착공 미이행 시 해지 가능 조항
계약서 2부 작성 (쌍방 보관) 1부만 있으면 분쟁 시 불리
플랫폼 통해 계약 시 업체명 일치 확인 오늘의집 등록명과 실제 업체명 대조
위약금 조항 쌍방 공정한가 소비자만 위약금 내는 조항은 부당

4 시공 중 관리

카카오톡 채팅 내용 수시 백업 팀채팅 강퇴당하면 증거 소실
통화 시 녹취 구두 약속은 증거 안 됨
매일 현장 사진 요청/촬영 앱이 없으면 카톡으로라도 받기
자재 반입 시 브랜드·모델 확인 견적서와 다른 자재 반입 시 즉시 항의
중도금은 공정 진행률에 맞춰 지급 공사 안 했는데 중도금 요구 시 거부 가능
변경 요청은 반드시 서면(카톡)으로 "말로 했는데 반영 안 됐다" 분쟁 방지

🚨 이런 신호가 보이면 즉시 재검토하세요

⚠️ 중도금을 대표 개인계좌로 입금 요청
⚠️ 도면·공정표 없이 철거 시작하려 함
⚠️ 마감재 샘플 미팅 없이 공사 진행
⚠️ "까다로운 고객" 운운하며 다른 업체 가라고 함
⚠️ 일정 컴플레인 시 "계약취소" 먼저 꺼냄
⚠️ 가족 직장 명함 요구 또는 협박성 발언
⚠️ 카카오톡 팀채팅에서 강퇴당함
⚠️ 신설 1년 미만 법인인데 포트폴리오가 화려
⚠️ 구청 허가 없이 확장공사 진행하려 함
⚠️ "추가금 없다"고 말만 하고 서면 거부
