# 얼마드나 — 전 공정 세부 설계서

> 각 공정별 칩 토글 구조, 자재/브랜드 선택지, 등급 연동 기본값, 단가 근거
> 작성일: 2026.04.06
> 원칙: "등급 선택 → 자동 세팅 → 원하면 커스텀"

---

## 설계 원칙

### 2단계 구조
1. **자동 세팅**: 사용자가 등급(기본/중급/고급)만 고르면 모든 공정의 세부 자재가 자동 배정
2. **커스텀**: 특정 공정을 탭하면 세부 자재/브랜드/수량을 직접 변경 가능. 변경 즉시 총액 반영.

### UI 패턴 (모든 공정 공통)
```
┌─ [공정명] ─────────── [소계 금액] ──┐
│                                     │
│  [세부항목 칩 토글 — ON/OFF]          │
│  [자재/브랜드 선택 — 라디오 또는 칩]   │
│  [수량 — 드롭다운 ▾]                 │
│  [자동 산출된 단가 × 수량 = 소계]     │
│                                     │
└─────────────────────────────────────┘
```

### 등급별 기본 컨셉
- **기본**: 가성비. 국산 보급형. 원룸/빌라/단기거주. 최소 비용.
- **중급**: 대중적 선택. 대림/LX/영림 등 국산 중견. 대부분의 30평대 아파트.
- **고급**: 프리미엄. 아메리칸스탠다드/수입/브랜드. 하이엔드 리모델링.

## 공정 1: 철거

### 세부 항목 (칩 토글)
```
[전체 철거✓] [부분 철거(도배+바닥만)]

전체 철거 선택 시 하위 항목 자동 ON:
  [바닥 철거✓]    기존: [장판] [마루] [타일] ← 라디오
  [도배 철거✓]    한지/곰팡이 제거 포함
  [욕실 올철거✓]  ▾ 1개소 / 2개소
  [주방 철거✓]    싱크대+벽타일
  [가구 철거✓]    붙박이장+신발장
  [몰딩 철거✓]    기존 몰딩/걸레받이/등박스
  [기타 철거]      에어컨배관/가스배관 등
```

### 단가 기준
| 항목 | 단가 | 단위 | 근거 |
|---|---|---|---|
| 장판 철거+샌딩 | 20,000 | 평 | EST-017 |
| 마루 철거+샌딩 (합판) | 23,000~35,000 | 평 | EST-007, EST-017 |
| MDF 철거 | 55,000 | 평 | EST-007 |
| 욕실 올철거 (벽+바닥+방수) | 900,000 | 개소 | EST-019 |
| 주방 철거 | 400,000~600,000 | 1식 | EST-025 추정 |
| 가구 철거 (전문팀) | 300,000 | 1식 | EST-019 |
| 내부철거 인건비 | 250,000 | 1일 | EST-025 |
| 폐기물 처리 | 400,000 | 1회(1톤) | EST-025 |
| 전체 올철거 인건비 | 250,000 × 4~6일 | | EST-025 |

### 등급 연동
- 기본: 부분 철거 (도배+바닥만) → 약 50~100만
- 중급: 전체 철거 (욕실2+주방+가구+바닥) → 약 200~300만
- 고급: 전체 철거 + 벽체 일부 해체 → 약 300~450만

### 특수 조건
- 주거유형 "신축 입주" 선택 시 → 철거 자동 OFF (또는 "부분 철거"만 활성)

### 목조 vs 경량스터드 비교
| 구분 | 목조(각목+합판) | 경량스터드(LGS) |
|---|---|---|
| 자재비 | 저렴 | 약 20~30% 비쌈 |
| 시공속도 | 보통 | 빠름 (규격화) |
| 직선 정밀도 | 기술자 의존 | 높음 (공장 규격) |
| 곡면/비정형 | 가능 | 어려움 |
| 습기 | 변형 가능 | 강함 |
| 방음 | 보통 | 그라스울 충진 시 우수 |
| 사용처 | 일반 주거 대부분 | 상업공간, 고급 주거 |

→ 얼마드나에서는 "가벽" 항목에서 [목조] [경량스터드] 선택 가능하게.
→ 경량스터드 선택 시 자재비 20~30% 추가 반영.
