## 1. 프로젝트 개요

### 1.1 서비스 정의

**얼마드나** (ulmadna.com) — 회원가입/개인정보 없이 바로 쓰는 무료 인테리어 예상 견적 계산기

운영 주체: 얼마드나 (독립 브랜드로 운영)

### 1.2 핵심 원칙

1. **완전 무료** — 모든 기능 비회원 100% 이용 가능
2. **개인정보 무수집** — 회원가입, 이메일, 전화번호 일절 없음
3. **원페이지 실시간** — 스텝퍼(Wizard) UI 금지, 입력 변경 즉시 결과 업데이트
4. **투명성** — 단가 근거, 계산 로직, 계수를 숨기지 않음
5. **비침해적 수익** — 팝업/모달/상담유도 버튼 없음, 배너 광고만

### 1.3 포지셔닝

> "견적 숫자를 보여주는 계산기"가 아니라, **"이 견적이 합리적인지 판단해주는 도구"**

| 구분 | 기존 서비스 (내공, 집닥, 오늘의집) | 얼마드나 |
|------|------|------|
| 목적 | 리드 수집 → 업체 매칭 | 소비자 정보 제공 (무료) |
| 개인정보 | 필수 수집 | 제로 |
| 자재 등급 | 미분리 | 기본/중급/고급 3단계 |
| 합리성 판단 | 없음 | 시장 평균 대비 배지 + 코멘트 |
| PDF 리포트 | 없음 | 즉시 무료 다운로드 |
| 지역/유형 보정 | 없음 | 4개 권역 × 3개 유형 × 4개 천장높이 |

#### FR-02: 계산 엔진

```
공정별 금액 = 기본 단가[등급] × (평형 / 30) × 지역계수 × 주거유형계수 × 천장높이계수
총 소계 = Σ(ON 상태인 공정의 금액)
예비비 = 소계 × 0.15
총 예산 = 소계 + 예비비
평당 단가 = 총 예산 / 평형
```

- 30평 기준 단가 DB를 Excel→JSON 변환하여 프론트엔드에 내장
- 입력 변경 시 즉시(실시간) 재계산 — 별도 "계산하기" 버튼 없음
- 에어컨은 등급별 분기: 기본=벽걸이 2대(90만), 중급/고급=시스템

### 3.1 데이터 흐름

```
[Excel 단가 DB] → (빌드 타임 변환) → [JSON 정적 데이터]
                                          ↓
[사용자 입력] → [계산 엔진 (클라이언트)] → [결과 렌더링]
                                          ↓
                              [PDF/엑셀 생성 (클라이언트)]
                                          ↓
                              [카카오 공유 (OG 태그)]
```

- Phase 1 핵심 계산은 100% 클라이언트 사이드 (서버 API 호출 없음)
- 단가 JSON은 빌드 타임에 생성되어 번들에 포함
- 회원 기능(저장/불러오기)만 서버 API 필요
