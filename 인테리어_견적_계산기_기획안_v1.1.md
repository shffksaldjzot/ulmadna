# 인테리어 예상 견적 계산기 — 서비스 기획안 v1.1

> **서비스명:** 얼마드나 (ulmadna.com)
> **작성일:** 2026.04.05
> **운영:** 도토리스튜디오
> **포지셔닝:** "견적 숫자를 보여주는 계산기"가 아니라, **"이 견적이 합리적인지 판단해주는 도구"**

## 1. 시장 현황 & 경쟁 분석

### 1-1. 기존 서비스 비교 분석

**내공 (naegong.com)**
아임웹 기반 노코드 사이트. 평형 입력 → 공정 체크 → 총액 산출의 단순 계산기 구조. 장인 찾기(공정별 수동 리스트), 컨설팅(전화 5만/대면 30만), 공사 후기 게시판, 캘린더 기능 제공. 올해 초 통신판매업 등록한 초기 서비스로, 판교 제2테크노밸리 경기스타트업브릿지 입주 중. 대표 김미현(서울인테리어협회 이사). 고객센터가 개인 핸드폰이라는 점에서 운영 규모가 작음.

**집닥 (zipdoc.co.kr)**
인테리어 비교견적 중개 플랫폼. 견적 계산기는 "영수증 기반 예상 비용" 방식으로, 건물 유형 → 상세 공사항목 선택 → 범위(min~max) 산출 구조. 핵심은 계산기가 아니라 견적 상담 신청 → 시공전문가 매칭. 계산기는 리드 생성 도구 역할.

**오늘의집 (ohou.se)**
"종합 시공 견적 계산기" 운영. 국내 최대 인테리어 플랫폼답게 방대한 시공 데이터 보유. 계산기 → 시공 전문가 연결 구조. 브랜드 파워와 트래픽에서 압도적이나, 개인 사용자 입장에서 견적의 "합리성 판단"까지는 해주지 않음.

**하우스텝 (houstep.co.kr)**
"아파트 인테리어 비용 계산기"라는 직관적 타이틀. 개별 공사~전체 공사 커버. 비교적 심플한 UI.

### 1-2. 공통적인 한계 (= 우리의 기회)

모든 기존 서비스에서 공통적으로 빠져있는 것들이 있다.

첫째, **단가 근거의 불투명성.** 어디서 온 숫자인지, 언제 기준인지 밝히지 않는다. "시장 평균"이라고만 할 뿐 조달청 단가, 실제 시공 데이터 등 구체적 근거가 없다.

둘째, **자재 등급 분리의 부재.** 합지 도배와 수입 벽지 도배를 같은 "도배"로 묶어서 보여준다. 사용자 입장에서 자기 상황에 맞는 현실적 금액을 알 수 없다.

셋째, **신축/구축 구분 미비.** 신축 입주 인테리어와 20년 된 아파트 리모델링은 공정 범위와 비용이 완전히 다른데, 이 차이를 반영하지 않는다.

넷째, **"그래서 이 금액이 비싼 거야, 싼 거야?"에 대한 답이 없다.** 숫자를 보여줄 뿐 해석을 해주지 않는다. 일반 소비자가 가장 궁금한 건 숫자 자체가 아니라 "합리적인가?"이다.

다섯째, **견적서 비교 기능의 부재.** 업체에서 받은 실제 견적서와 시장 평균을 나란히 비교하는 기능이 없다.

여섯째, **전부 리드 수집 목적의 CTA 중심 구조.** 집닥은 결과를 보려면 상담 신청이 필요하고, 오늘의집은 전문가 매칭으로 유도하며, 내공도 컨설팅 전환을 전제로 설계되어 있다. 소비자 입장에서 "결국 영업이구나"라는 인상을 주며, 인테리어 업계에 대한 기존 불신을 강화한다. 회원가입, 전화번호 입력, 상담 신청 없이 순수하게 쓸 수 있는 견적 도구가 시장에 없다.

### 1-3. 시장 기회

인테리어 시장에서 정보 비대칭은 여전히 심각하다. 소비자는 "30평 리모델링 얼마야?"를 검색하고, 블로그마다 다른 숫자에 혼란을 겪는다. 계산기는 이 혼란을 해결하는 첫 번째 접점이 되는데, 기존 서비스들은 이 접점에서 개인정보를 수집하려 하기 때문에 소비자의 경계심을 오히려 높인다. **"진짜 무료, 진짜 부담 없는" 도구**가 이 시장에서 신뢰를 독점할 수 있는 기회다.

핵심 타깃은 **"직접 알아보는 30~40대 신규 아파트 입주자 및 리모델링 계획자"**이다.

## 2. 제품 전략

### 2-1. 핵심 가치 제안 (Value Proposition)

> **"견적 숫자가 아니라, 견적의 의미를 알려드립니다."**
> **"회원가입 없음. 전화번호 없음. 그냥 쓰세요."**

단순 계산기가 아니다. 사용자가 입력한 조건에 맞는 예상 견적을 산출하되, 그 금액이 시장 대비 어떤 위치인지, 어디에서 비용을 절약할 수 있는지, 어떤 순서로 공사해야 하는지까지 안내하는 **"견적 리포트 도구"**다.

핵심 원칙은 **완전 무료 + 개인정보 무수집**이다. 회원가입, 로그인, 이메일, 전화번호 — 아무것도 요구하지 않는다. PDF 다운로드도 바로 받을 수 있다. 이 원칙 자체가 경쟁사 대비 가장 강력한 차별화 포인트이며, 커뮤니티 바이럴의 핵심 메시지("여기는 전화번호 안 넣어도 되는 견적 계산기")가 된다.

### 2-2. 서비스명

**얼마드나** (ulmadna.com)

"인테리어 얼마 드나"라는 자연어 검색 패턴과 정확히 일치한다. 네이버·유튜브에서 소비자가 실제로 검색하는 문장이 그대로 서비스명이 되므로 SEO 효과가 극대화된다. 발음이 쉽고 한 번 들으면 기억에 남으며, 도메인(ulmadna.com)도 확보 완료.

### 2-3. 핵심 기능 구조

**[Phase 1] 입력 → 산출 → 리포트**

사용자 입력 5단계:

1단계 — **평형 선택:** 10평 / 20평 / 24평 / 30평 / 34평 / 40평 / 50평 / 직접입력
2단계 — **주거 유형:** 신축 입주 / 구축 리모델링(10년 미만) / 구축 리모델링(20년 이상)
3단계 — **지역:** 서울 / 경기·인천 / 지방광역시 / 기타 지방
4단계 — **자재 등급:** 기본(가성비) / 중급(대중적) / 고급(프리미엄) — 각 등급별 대표 브랜드·자재명 예시 표시
5단계 — **공정 선택:** 14개 공정 카테고리별 ON/OFF 토글 + 세부 옵션(예: 욕실 개수, 에어컨 대수)

산출 결과:

(a) **공정별 상세 견적표** — 항목 / 수량 / 단가 / 소계가 표 형태로 보임
(b) **총 예상 비용 범위** — 최저~최고 밴드로 표시, 중앙값 강조
(c) **비용 비중 차트** — 도넛 또는 수평 바 차트로 "돈이 가장 많이 가는 곳" 시각화
(d) **합리성 판단 코멘트** — "이 금액은 시장 평균 대비 [적정/다소 높음/매우 합리적] 수준입니다" + 항목별 한 줄 코멘트
(e) **예비비 권장** — 총 예산의 15~20% 자동 산출
(f) **공정 순서 + 예상 기간** — 타임라인 형태로 "어떤 공정이 먼저이고 전체 몇 주 걸리는지"
(g) **PDF 리포트 다운로드** — 위 내용을 깔끔한 A4 리포트로 출력

**[Phase 2] 견적서 비교 기능 (v2)**

사용자가 업체에서 받은 실제 견적서를 업로드(또는 항목별 수동 입력)하면, 시장 평균 단가와 나란히 비교하는 기능. 항목별로 "이 단가는 시장 대비 높음/적정/낮음" 표시. 이 기능 역시 회원가입 없이 무료로 제공하며, 서비스의 전문성과 신뢰도를 높이는 핵심 콘텐츠 역할을 한다.

**[Phase 3] AI 견적 분석 (v3)**

업로드된 견적서를 AI가 읽어서 자동 분류하고 시장 단가와 비교 분석. 항목 누락 여부 감지, 중복 청구 항의, 단가 이상치 경고 기능. 이 단계에서 Signnote 연동도 고려.

## 3. 단가 데이터베이스 설계

### 3-1. 데이터 구조

모든 단가는 다음 차원으로 분류한다:

**공정(14개 카테고리)** × **세부 항목** × **자재 등급(기본/중급/고급)** × **지역 계수** × **주거 유형 계수**

지역 계수: 서울 1.0 / 경기·인천 0.92 / 지방광역시 0.85 / 기타 지방 0.78
주거 유형 계수: 신축 입주 0.7 / 구축 10년 미만 0.9 / 구축 20년 이상 1.0

→ 구축 20년 이상이 기준값(1.0)이고, 신축은 철거·배관 등이 불필요하므로 0.7 적용

### 3-2. 공정별 단가표 (2025~2026 기준, 30평 구축 20년+ 서울 기준)

**① 철거**

전체 철거는 평당 기본 5만~고급 15만원 수준이며 폐기물 처리비를 포함한다. 부분 철거(도배+바닥)는 평당 3만~7만원, 욕실 철거는 개소당 30만~70만원, 주방 철거는 싱크대 포함 개소당 30만~80만원이다. 폐기물 처리는 5톤 기준 별도 30만~50만원.

**② 전기**

전체 전기 공사는 30평 기준 1식 80만~250만원 범위. 세부로는 콘센트 이동/추가 개당 3만~7만, 스위치 교체 개당 1.5만~5만, 분전반 교체 1식 15만~40만, 다운라이트 배선 개당 2만~5만, 간접조명 배선 m당 5만~15만원.

**③ 설비(배관)**

급수·배수 배관 교체 1식 80만~180만, 난방 배관 교체 1식 100만~200만, 화장실 배관 이동 개소당 50만~120만, 주방 배관 이동 1식 40만~100만원.

**④ 보일러**

국산 보일러 교체 1식 80만~180만, 콘덴싱 보일러 1식 120만~250만원.

**⑤ 에어컨**

벽걸이 에어컨 대당 40만~80만, 시스템 에어컨 3개소 1식 350만~650만, 5개소 1식 550만~950만, 매립 배관만 개소당 15만~30만원.

**⑥ 목공**

전체 목공은 30평 기준 1식 200만~700만원 범위. 천장 평탄화 평당 3만~8만, 우물천장 개소당 15만~40만, 몰딩 m당 0.5만~2만, 걸레받이 m당 0.3만~1.5만, 가벽 설치 m당 10만~35만, 문틀+문짝 교체 개당 15만~40만, 아트월 1식 30만~150만원.

**⑦ 도장**

전체 도장 30평 기준 1식 80만~250만원. 벽면 평당 1.5만~4만, 천장 평당 1만~3만원.

**⑧ 도배**

전체 도배 30평 기준 1식 55만(합지)~150만(수입)원. 평당으로는 합지 1.8만, 실크 3만, 수입 5만원.

**⑨ 바닥**

강화마루 평당 3만~8만, 강마루 평당 5만~12만, 원목마루 평당 10만~25만, 장판 평당 1.5만~5만, 데코타일 평당 3.5만~8만원.

**⑩ 욕실**

전체 리모델링 개소당 250만~800만원. 방수 공사 개소당 30만~80만, 도기(세면대+변기) 개소당 30만~120만원. 타일은 등급별 리모델링 비용에 포함.

**⑪ 주방**

싱크대 교체는 상판에 따라 MMA 150만, 엔지니어드스톤 300만, 세라믹 500만원. 주방 벽타일 1식 30만~100만, 빌트인 가전(쿡탑+후드) 1식 50만~250만원.

**⑫ 조명**

전체 조명 30평 기준 1식 50만~250만원. LED 등기구 개당 3만~15만, 다운라이트 개당 1.5만~5만, 간접조명 m당 3만~12만원.

**⑬ 가구·기타**

중문 개당 30만~150만, 붙박이장 개당 50만~250만, 신발장 1식 30만~120만, 커튼 전 세대 1식 30만~200만, 필름 평당 1만~3.5만, 창호(샷시) 교체 평당 15만~40만원.

**⑭ 부대비용**

보양(엘리베이터+복도) 20만~50만, 준공 청소 15만~30만, 입주 청소 15만~30만, 감리비(선택) 50만~150만원.

### 3-3. 단가 보정 방침

이 단가표는 리서치 기반 시장 평균값이므로, 반드시 실무 경험치로 보정해야 한다. 특히 목공, 욕실, 주방은 세부 사양에 따른 편차가 크므로 구간을 더 세분화할 필요가 있다. 단가 업데이트는 반기 1회 이상 진행하고, 조달청 적산 자료와 실제 시공 데이터를 교차 검증한다.

## 5. UX 설계 방향

### 5-1. 사용자 여정 (User Flow)

```
[랜딩 페이지]
    ↓ "바로 견적 뽑아보기" (로그인·가입 없음)
[Step 1] 평형 선택 (카드 UI, 원터치)
    ↓
[Step 2] 주거 유형 선택 (일러스트 + 설명)
    ↓
[Step 3] 지역 선택 (지도 UI)
    ↓
[Step 4] 자재 등급 선택 (비교 카드: 브랜드 예시 포함)
    ↓
[Step 5] 공정 선택 (토글 + 세부옵션)
    ↓
[결과 페이지]
    ├─ 공정별 견적표
    ├─ 비용 비중 차트
    ├─ 합리성 판단 코멘트
    ├─ 공정 순서 타임라인
    ├─ [PDF 다운로드] → 바로 다운 (개인정보 입력 없음)
    ├─ [카카오톡 공유] → 결과 카드 이미지 공유
    ├─ 협력 브랜드 배너 (2~3개, 비침해적 위치)
    └─ "도토리스튜디오가 만들었습니다" (하단 소개, 링크만)
```

### 5-2. UI 원칙

**부담 제로 경험.** 서비스 진입부터 결과 확인, PDF 다운로드까지 전 과정에서 개인정보를 일절 요구하지 않는다. 팝업, 모달, 상담 유도 버튼 없음. 랜딩페이지 상단에 "회원가입 없이 바로 사용 · 전화번호 입력 없음 · 완전 무료"를 명시하여 기존 서비스에 실망한 사용자의 심리적 장벽을 제거한다.

**모바일 퍼스트.** 인테리어 검색의 70% 이상이 모바일에서 이루어진다. 한 화면에 한 질문, 큰 터치 타겟, 진행률 표시 바.

**신뢰감 디자인.** 숫자가 어디서 왔는지 투명하게 보여준다. "2025~2026 시장 평균 기반, 조달청 적산 교차 검증"이라는 문구를 결과 하단에 명시. 개인정보를 안 받는다는 점 자체가 가장 강력한 신뢰 신호다.

**즉시 피드백.** 공정을 추가/제거할 때마다 총 예상 금액이 실시간 업데이트. 사용자가 조합을 바꿔보며 예산을 탐색할 수 있게 한다.

**결과 페이지의 정보 계층.** 첫 화면에 보이는 건 총 금액 + 한 줄 판단("적정 수준입니다"). 스크롤하면 상세 견적표, 차트, 코멘트가 순서대로 나타남.

## 6. 수익 모델 & 비즈니스 연결

### 6-1. 핵심 원칙: "사용자에게는 무료, 수익은 B2B에서"

이 서비스는 소비자에게 어떤 비용도, 어떤 개인정보도 요구하지 않는다. 수익은 전적으로 광고·제휴·브랜드 스폰서십에서 발생한다. 소비자가 부담 없이 쓸수록 트래픽이 늘고, 트래픽이 늘수록 광고 지면의 가치가 올라가는 구조다.

### 6-2. 수익 채널

**1차 — 자재 브랜드 스폰서십 (핵심)**

자재 등급 선택 화면에서 브랜드가 자연스럽게 노출된다. 예를 들어 "중급 — 실크 벽지 (예: LG하우시스 디아망)" 식으로 등급 설명에 실제 브랜드·제품명이 함께 표시된다. 결과 페이지의 공정별 견적표에서도 "욕실 타일 — 포세린 600×600 (예: 대림바스 아넬로)" 형태로 노출 가능.

과금 방식은 월정액(지면 고정 노출) 또는 CPC(클릭당 과금). 타깃 브랜드: 한샘, LG하우시스, 대림바스, 현대L&C, KCC글라스, 노루페인트, 삼화페인트, 동화자연마루 등.

**2차 — 결과 페이지 배너 광고**

결과 페이지 하단에 협력업체 배너 2~3개를 비침해적으로 배치한다. 콘텐츠 영역(견적표, 차트, 코멘트)이 끝난 후 하단에 위치시켜 사용자 경험을 방해하지 않는다. 사용자가 선택한 지역 기반으로 타깃팅하면 광고 단가가 올라간다. 예를 들어 "경기 지역" 선택 시 경기 소재 자재 매장이나 시공 업체 배너가 노출되는 식.

과금 방식은 CPM(노출당) 또는 CPC. 초기에는 도토리스튜디오와 기존 협력 관계가 있는 업체부터 시작.

**3차 — 도토리스튜디오 자연 유입**

결과 페이지 최하단에 "이 계산기는 10년 경력 인테리어 전문가가 만들었습니다 — 도토리스튜디오" 형태의 소개를 작게 넣는다. 강제 CTA가 아니라 브랜드 크레딧 수준. 관심 있는 사람이 스스로 찾아오는 구조. 이 서비스를 꾸준히 쓰는 사용자일수록 "이걸 만든 곳이면 믿을 만하겠다"는 인식이 자연스럽게 형성된다.

### 6-3. 수익 시뮬레이션

월 방문자 5,000명 기준(런칭 3~6개월 후 목표):

자재 브랜드 스폰서 2~3개 × 월 50만~100만원 = 월 100만~300만원
결과 페이지 배너 CPM ₩3,000 기준 × 5,000회 = 월 15만원 (트래픽 증가에 따라 성장)
도토리스튜디오 컨설팅 자연 유입 월 2~3건 × 35만원 = 월 70만~105만원

**초기 월 매출 예상: 약 185만~420만원**

트래픽이 월 50,000명 수준으로 성장하면(SEO + 유튜브 효과) 배너 광고만으로 월 150만원, 스폰서십 단가 인상으로 월 500만~1,000만원대 수익 가능.

### 6-4. 광고 운영 원칙

광고는 반드시 아래 원칙을 지킨다:

(a) 콘텐츠 영역 안에 광고를 끼워넣지 않는다 — 견적표, 차트, 코멘트 사이에 배너 삽입 금지
(b) 자재 브랜드 노출은 "예시" 형태로만 — "이 브랜드를 써야 한다"는 식의 추천 금지
(c) 모든 제휴 노출에 "제휴" 또는 "광고" 표시를 명확히 한다
(d) 팝업, 모달, 풀스크린 광고 절대 사용하지 않는다
(e) 광고 때문에 페이지 로딩이 느려지면 안 된다

## 7. 콘텐츠 마케팅 & 트래픽 전략

### 7-1. SEO 전략

**핵심 키워드 타깃:**
"인테리어 견적", "인테리어 비용", "30평 리모델링 비용", "아파트 인테리어 평당 가격", "인테리어 견적 계산기", "인테리어 견적서 보는 법"

**콘텐츠 허브 구조:**
계산기 자체가 링크 대상이 되는 SEO 자산. 블로그에서 "30평 인테리어 비용 총정리" 글을 쓰고, 글 안에서 "직접 계산해보기 →" 링크로 계산기에 유입시키는 구조.

### 7-2. 유튜브/인스타그램 연동

촬영 콘텐츠에서 자연스럽게 계산기를 활용한다.

"이 30평 아파트, 실제 공사비가 얼마였을까요?" → 계산기로 예상 견적 뽑아보고 → 실제 금액과 비교 → "차이가 나는 이유는..." → "여러분도 직접 뽑아보세요, 링크 프로필에"

이 포맷은 리얼리티쇼 느낌으로 반복 가능하며, 시청자 참여를 유도한다.

### 7-3. 바이럴 요소

가장 강력한 바이럴 메시지는 **"여기는 전화번호 안 넣어도 됨"**이다. 인테리어 준비 중인 사람들이 모이는 네이버 카페(예: 아파트스토리, 인스, 오하우), 맘카페, 부동산 커뮤니티에서 이 한 마디가 공유 이유가 된다. 기존 견적 계산기에 지친 사용자가 "드디어 정상적인 곳 찾음"이라고 추천해주는 구조.

견적 결과 페이지에 **"내 견적 공유하기"** 버튼. 카카오톡/블로그/커뮤니티에 결과 카드 이미지가 공유된다. "30평 중급 리모델링 예상 견적: 약 4,255만원 — 얼마드나에서 확인" 형태의 공유 카드.

## 8. 기술 구현 로드맵

### Phase 0: MVP 검증 (1~2주)

구글시트로 계산 로직을 먼저 구현. 시트 링크를 공개하여 누구나 "사본 만들기"로 자기 조건을 입력해볼 수 있게 한다. 네이버 카페나 지인 그룹에 공유하여 "사람들이 정말 쓰는가?", "어떤 공정을 가장 많이 궁금해하는가?"를 검증. 이 단계에서 개인정보를 수집하지 않으므로 구글폼은 불필요.

### Phase 1: 웹앱 v1 (4~6주)

React 기반 싱글 페이지 웹앱. 스텝별 입력 UI + 결과 페이지 + PDF 다운로드. 단가 데이터는 JSON 파일로 관리. 호스팅은 Vercel 또는 Cloudflare Pages(무료).

기술 스택: React + TypeScript + Tailwind CSS + Recharts(차트) + jsPDF(PDF 생성)

### Phase 2: 견적 비교 + 광고 시스템 (8~12주)

견적서 비교 기능 추가 (회원가입 없이 이용). 광고 관리 대시보드(관리자용: 배너 등록, 노출 통계, 클릭 추적). 단가 데이터 CMS화(관리자가 직접 수정 가능). 트래픽 분석 연동(GA4, 네이버 서치어드바이저).

백엔드 추가: NestJS + PostgreSQL (Signnote 스택과 통일)

### Phase 3: AI 분석 + 플랫폼화 (6개월+)

견적서 이미지/PDF 업로드 → OCR + AI 분석. Signnote 연동(계약-결제 흐름). 시공 업체 매칭(선택적).

## 10. 리스크 & 대응

**리스크 1 — 단가 정확도 논란**
대응: "참고용 예상 견적"임을 명시. 실제 시공비는 현장 실측 필요하다는 면책 문구 상시 표시. 단가 출처(조달청, 시장 조사)를 투명하게 공개.

**리스크 2 — 대형 플랫폼의 기능 추격**
대응: 오늘의집·집닥은 "매칭 플랫폼"이 본업이므로, 계산기에 깊이를 투자할 유인이 적음. 우리는 계산기 자체가 핵심 제품이므로 디테일에서 이길 수 있음.

**리스크 3 — 트래픽 확보 실패 → 광고 수익 부족**
대응: 광고 수익은 트래픽에 비례하므로 초기에는 수익이 작을 수 있다. 하지만 이 서비스의 1차 목적은 "도토리스튜디오의 브랜드 자산 구축"이지, 단독 수익 사업이 아니다. 무료 도구 + SEO + 유튜브 콘텐츠의 삼각 구조로 트래픽을 꾸준히 키우되, 초기에는 네이버 카페·커뮤니티 바이럴로 씨앗 트래픽 확보. 도토리스튜디오 기존 고객에게 먼저 테스트.

**리스크 4 — 광고가 신뢰도를 훼손할 가능성**
대응: 광고 운영 원칙(6-4항)을 엄격히 준수. 콘텐츠 안에 광고를 끼워넣지 않고, 모든 제휴에 "광고" 표시를 명확히 한다. 사용자 피드백을 모니터링하여 광고가 부담스럽다는 반응이 나오면 즉시 배치를 조정한다.
