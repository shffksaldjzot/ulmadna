---

## 1. 지역 타겟 시스템

### 위치 판별 방식
- **1차**: 사용자 IP 주소 기반 자동 감지 (시/구/동 단위)
- **2차**: 서비스 내 "지역" 선택값 (사용자가 직접 선택한 지역 우선)
- IP 감지 실패 시 → "지역: 서울" 기본값 적용

### 지역권 분류
```
수도권 (55개 권역):
  서울 25개구
  경기 31개 시군 → 주요 10개 묶음 (성남/용인/화성/고양/남양주/파주/김포/수원/안양/평택)

광역시 (7개 권역):
  부산 / 대구 / 인천 / 광주 / 대전 / 울산 / 세종

도 단위 (9개 권역):
  경기외곽 / 강원 / 충북 / 충남 / 전북 / 전남 / 경북 / 경남 / 제주
```

→ **같은 배너 자리에 지역별로 다른 광고주**를 보여줌
→ 사용자는 배너 1개만 보이고, 우리는 지역별 광고주 각각에게 비용을 받음

## 2. 배너 상품 종류 (4가지)

### A. 서비스 페이지 배너
서비스(견적 계산기) 페이지 내에 직접 노출되는 배너

### B. 공정별 스폰서 배너
특정 공정을 선택/열었을 때 해당 공정 내에 노출되는 브랜드 배너

### C. 결과 페이지 배너
견적 결과 요약 영역(우측 패널 또는 모바일 하단)에 노출

### D. PDF/엑셀 내 광고
다운로드 견적서 하단에 포함되는 추천 업체 리스트

## 5. 배너 디자인 규칙

### 공통 규칙
- 배너 테두리: 1px solid #E0D6CC (따뜻한 베이지 계열, 페이지와 조화)
- 배너 배경: 흰색 또는 크림(#F5F0EB) — **절대 원색/형광 배경 금지**
- border-radius: 8px (모서리 둥글게)
- 배너 상단 좌측에 **"AD"** 또는 **"광고"** 라벨 필수 (6px, #999)
- 배너와 콘텐츠 사이 여백: 최소 16px

### 배너 내부 콘텐츠 가이드 (광고주에게 제공)
```
┌─────────────────────────────────────┐
│ 광고                                 │ ← "광고" 라벨 (6px, #999)
│                                     │
│  [로고]  업체명 / 브랜드명            │ ← 좌측 로고, 우측 텍스트
│          한 줄 카피                   │
│          [자세히 보기 →]              │ ← CTA 버튼 (선택)
│                                     │
└─────────────────────────────────────┘
```

### 금지 사항
- ❌ 팝업 광고 (절대 금지)
- ❌ 자동 재생 동영상 광고
- ❌ 전면 광고 (인터스티셜)
- ❌ 콘텐츠를 가리는 오버레이 광고
- ❌ 깜빡이는/애니메이션 과도한 배너
- ❌ 광고임을 숨기는 네이티브 광고 (반드시 "광고" 라벨 표시)

## 6. 배너 슬롯별 상세 스펙

### AD-H (헤더 하단 리더보드) — 728×90

```
┌──────────────────────────────────────────────────┐
│ 광고                                              │
│                                                  │
│  [🏠 로고]  ○○인테리어 — 남양주 No.1 시공 전문     │
│            무료 상담 1588-XXXX  [견적 받기 →]      │
│                                                  │
└──────────────────────────────────────────────────┘
728px
```
- **타겟**: 지역권 (사용자 IP 기반)
- **광고주 예시**: 해당 지역 인테리어 업체, 자재 대리점
- **최대 광고주 수**: 지역권 수만큼 (서울 25 + 경기 10 + 광역시 7 + 도 9 = 약 51개)
- **노출 규칙**: 1회 방문당 1개 광고주만 노출 (로테이션 없음)

### AD-P1, AD-P2 (공정 사이 슬림 배너) — 468×60

```
┌───────────────────────────────────────┐
│ 광고  [LX Z:IN 로고] 창호는 LX로 — 에너지등급 1등급 보장  [자세히 →] │
└───────────────────────────────────────┘
468px
```
- **AD-P1 위치**: 철거~목공~바닥 사이 (상단 공정 그룹 후)
- **AD-P2 위치**: 욕실~주방~가구 사이 (중단 공정 그룹 후)
- **타겟**: 공정별 스폰서 (예: 창호 공정 근처에 KCC/LX 배너)
- **광고주 예시**: LX하우시스, KCC, 대림바스, 영림도어, 루씨에어 등
- **최대 광고주 수**: 공정별 1개씩 = 24개 공정 × 1 = 24개
