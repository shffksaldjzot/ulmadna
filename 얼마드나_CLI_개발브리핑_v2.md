## 2. UI 패턴: 원페이지 실시간 계산기

스텝퍼(Wizard) 금지. 원페이지에서 모든 입력과 결과가 보이고 옵션 변경 시 결과 즉시 업데이트.

**PC:** 입력55~60% : 결과40~45%. 결과 패널 sticky.
**모바일:** 기본조건 칩 압축 + 총금액 상단 고정 + 공정별 리스트 탭→바텀시트.

## 3. 3단 Progressive Disclosure (핵심 설계)

### 1단계 (90% 사용자 — 3초)
기본 조건 4개만 → 총액:
- 평형: Radio [18/24/30/33/34/39/43/직접입력]
- 유형: Radio [신축입주/구축10년/구축20년+]
- 지역: Radio [서울/경기/지방/기타]
- 등급: Radio [기본/중급/고급]

### 2단계 (30% — 30초)
공정 탭하면 핵심 2~4개 칩. 브랜드명 안 보임.

### 3단계 (5% — 3분)
"상세설정▸" 누르면 브랜드/모델/세부옵션 전체.

**원칙:** 등급 변경→전체 세부 일괄 변경. 개별 커스텀→그것만 변경, 나머지 등급 기본값 유지.

### 입력 타입
- **Radio(●):** 하나만 선택 (자재종류, 브랜드, 등급)
- **Checkbox(☐):** 다중 선택 (추가옵션 — 파티션, 젠다이 등)
- **Stepper(▾):** 수량 (욕실개수, 도어조수, 에어컨대수)
- **Toggle(⊙):** ON/OFF (공정전체, 단일옵션)

### 계산 로직
```
공정별 금액 = Σ(선택옵션 price) × 수량 × 지역계수 × 유형계수
총 소계 = Σ(ON 공정)
예비비 = 소계 × 예비비율(5~15%)
총 예산 = 소계 + 예비비
```

### 계수표
서울1.00 / 경기0.92 / 지방0.85 / 기타0.78
신축0.70 / 구축10년0.90 / 구축20년+1.00
