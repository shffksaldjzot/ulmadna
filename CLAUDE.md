## 메인 규칙 (반드시 지킬 것)

1. **모든 코드에 한글 주석 필수** — 다른 개발자가 읽어도 바로 이해할 수 있도록 함수, 변수, 로직마다 한글 주석을 달 것
2. **모든 설명은 한글로** — 사용자는 한글만 사용하는 초보 개발자. 기술 용어는 쉽게 풀어서 설명할 것
3. **대규모 작업 전 반드시 계획안 보고 → 승인 후 진행** — 새 파일 3개 이상 생성, 기존 구조 변경, 엔진 로직 수정 등은 반드시 먼저 계획을 보여주고 승인받을 것
4. **날짜 포맷: YYYY년 MM월 DD일** (예: 2026년 04월 14일)
5. **API 키, 시크릿 하드코딩 금지** — 환경변수(.env)로만 관리. 절대 코드에 직접 쓰지 말 것
6. **모바일 최우선 (Mobile First)** — 모든 UI는 모바일 화면 기준으로 먼저 만들고, 데스크톱은 그 다음

## v3 핵심 구조

### 견적 뎁스 체계 (용어 기준)

| 뎁스 | 용어 | 설명 | 예시 |
|------|------|------|------|
| **0뎁스** | **평형** | 아파트 기본 정보 | 59㎡ / 74㎡ / 84㎡ |
| **1뎁스** | **등급** | 전체 프리셋 (한 번에 다 결정) | 가성비 / 중급 / 고급 |
| **2뎁스** | **공정** | 큰 공사 카테고리 (ON/OFF 가능) | 창호, 도배, 바닥, 욕실, 문... |
| **3뎁스** | **품목** | 공정 안의 개별 항목 (개별 수정 가능) | 거실창, 안방창, 침실창... |
| **4뎁스** | **사양** | 품목의 세부 선택 (선택사항) | 이중창/시스템창, 브랜드, 유리종류... |

### 사용자 흐름
```
[0뎁스] 평형 선택 (59/74/84 또는 직접입력)
    ↓
[1뎁스] 등급 선택 (가성비/중급/고급)
    ↓ → 2~4뎁스 전부 자동 세팅
    ↓
[2뎁스] 공정별 총액 확인 + ON/OFF
    ↓ → "세부 조정" 클릭 시
    ↓
[3뎁스] 품목별 개별 수정 (거실창만 시스템창, 침실2 교체안함 등)
    ↓ → 더 상세히 보기 시
    ↓
[4뎁스] 사양 선택 (브랜드, 유리종류 등)
```

- **초보 사용자**: 0뎁스 + 1뎁스만 고르면 끝 (2번 클릭)
- **좀 아는 사용자**: 2뎁스에서 공정 ON/OFF
- **전문 사용자**: 3~4뎁스에서 품목별 사양까지 직접 조정

### 계산 방식
```
입력: 타입(59/74/84) + Bay(2/3/4) + 구성 + 확장여부 + 등급 + 옵션
  → 62건 비율표 매칭 → 실별 물량 자동산출
  → 물량 x 공정별 단가
  → 실별/공정별 상세 견적서 출력
```

### 핵심 데이터 (절대 임의 수정 금지)
- **물량 DB**: 62건 평면도 분석 (59타입 25건, 74타입 16건, 84타입 21건)
- **단가 DB**: ulmadna_db.json (19개 공정, 35건 견적서 기반)
- **비율표 엑셀**: 얼마드나_59_74_84_통합비율표_v1.xlsx

### 절대 하지 말 것
- 비율표 데이터 임의 수정 금지 (도메인 전문가 검증 데이터)
- 단가 DB 값을 추정으로 채우지 말 것 (반드시 ulmadna_db.json 기준)
- "평당 단가 x 면적" 방식 재사용 금지
- 물량 표시 없이 금액만 출력하는 구조 금지

## 파일 구조 (v3 목표)

```
src/
├── data/
│   ├── quantity_db.json      ← 62건 물량 DB (비율표 JSON 변환)
│   └── price_db.json         ← 단가 DB (ulmadna_db.json 기반)
├── engine/
│   ├── matcher.ts            ← 프로필 매칭 로직
│   ├── calculator.ts         ← 물량 x 단가 계산
│   ├── formatter.ts          ← 결과 포맷팅
│   └── types.ts              ← 타입 정의
├── components/
│   ├── TypeSelector.tsx       ← STEP 1: 타입 선택
│   ├── StructureSelector.tsx  ← STEP 2: 구조 선택
│   ├── GradeSelector.tsx      ← STEP 3: 등급 선택
│   ├── OptionToggle.tsx       ← STEP 3: 옵션
│   ├── ResultCard.tsx         ← STEP 4: 결과 총액
│   ├── ProcessAccordion.tsx   ← STEP 4: 공정별 펼침
│   └── PDFDownload.tsx        ← PDF 생성
├── hooks/
│   ├── useEstimate.ts         ← 견적 계산 훅
│   └── useProfile.ts          ← DB 매칭 훅
└── pages/
    └── index.tsx              ← 메인 페이지
```
