# 얼마드나 (ulmadna.com) — Claude Code CLI 개발 브리핑

## 1. 프로젝트 개요

**서비스명:** 얼마드나
**도메인:** ulmadna.com
**한 줄 정의:** 회원가입·개인정보 입력 없이 바로 쓰는 무료 인테리어 예상 견적 계산기
**운영 주체:** 도토리스튜디오 (별도 서비스로 운영)

**핵심 원칙:**
- 완전 무료, 개인정보 무수집 (회원가입, 로그인, 이메일, 전화번호 일절 없음)
- PDF 다운로드도 바로 가능 (개인정보 입력 없이)
- 수익은 자재 브랜드 스폰서십 + 배너 광고로만 발생
- 팝업, 모달, 상담 유도 버튼 없음

---

## 2. UI 패턴: 원페이지 실시간 계산기

### ❌ 사용하지 않는 패턴
**스텝퍼(Stepper/Wizard) UI 사용 금지.** 숨고, 토스, 오늘의집처럼 한 단계씩 화면이 전환되는 방식을 쓰지 않는다.
- 매 단계마다 이탈률 발생
- "편하게 쓰세요" 콘셉트와 충돌
- 배너 광고 지면 확보 불가
- 결과를 보려면 끝까지 가야 하는 답답함

### ✅ 사용하는 패턴
**원페이지 실시간 계산기.** 모든 입력과 결과가 한 화면에 보이고, 옵션을 바꿀 때마다 결과가 즉시 업데이트된다.

**데스크톱 레이아웃:**
```
┌─────────────────────────────────────────────┐
│  헤더 (로고 + "완전 무료 · 개인정보 없음")    │
├──────────────────┬──────────────────────────┤
│                  │                          │
│   입력 패널      │    결과 패널              │
│   (좌측 고정)    │    (우측 스크롤)           │
│                  │                          │
│  ┌────────────┐  │  ┌──────────────────┐    │
│  │ 평형 선택  │  │  │ 총 예상 비용      │    │
│  │ 주거 유형  │  │  │ 합리성 판단       │    │
│  │ 지역      │  │  │ 공정별 견적표     │    │
│  │ 자재 등급  │  │  │ 비용 비중 차트    │    │
│  │ 공정 토글  │  │  │ 공정 순서 가이드  │    │
│  └────────────┘  │  │ [PDF 다운로드]    │    │
│                  │  │ [카카오톡 공유]    │    │
│                  │  │                    │    │
│                  │  │ ── 배너 광고 ──    │    │
│                  │  │ (협력 브랜드 2~3개) │    │
│                  │  │                    │    │
│                  │  │ 도토리스튜디오 소개 │    │
│                  │  └──────────────────┘    │
├──────────────────┴──────────────────────────┤
│  푸터                                        │
└─────────────────────────────────────────────┘
```

**모바일 레이아웃:**
```
┌───────────────────┐
│ 헤더              │
├───────────────────┤
│ 입력 영역         │
│ (아코디언/접기)    │
│ 평형·유형·지역·등급│
│ 공정 토글         │
├───────────────────┤
│ [견적 보기] 버튼   │
├───────────────────┤
│ 결과 영역         │
│ 총 예상 비용      │
│ 합리성 판단       │
│ 공정별 견적표     │
│ 비용 비중 차트    │
│ 공정 순서 가이드  │
│ [PDF] [공유]      │
│                   │
│ ── 배너 광고 ──   │
│                   │
│ 도토리스튜디오    │
├───────────────────┤
│ 푸터              │
└───────────────────┘
```

**핵심:** 입력 옵션을 바꿀 때마다 결과가 즉시(실시간) 업데이트. 별도 "계산하기" 버튼 없이 자동 반영. 모바일에서는 입력 영역을 접을 수 있게 하되, 결과는 항상 보이도록.

## 3. 디자인 가이드 (도토리스튜디오 브랜딩 연계)

| 메인 | 딥 체스트넛 브라운 | #5C3A21 | 헤더, 주요 텍스트, 강조 |
| 서브 배경 | 따뜻한 크림 | #F5F0EB | 페이지 배경, 입력 패널 배경 |

| 주의/보통 | 앰버 옐로 | #D4A017 | "적정 수준" 표시 |
| 안심/저렴 | 딥 그린 | #2E7D5B | "합리적" 표시, 절약 팁 |

### 타이포그래피
- 제목: Pretendard Bold 또는 Noto Sans KR Bold (고급감 + 가독성)
- 본문: Pretendard Regular / Noto Sans KR Regular
- 숫자/금액: Pretendard SemiBold, 크게 (금액은 사용자가 가장 먼저 보는 요소)

### 무드
- 고급 + 따뜻 + 전문적이지만 접근 가능한
- 깔끔하고 여백 많은 레이아웃
- 불필요한 장식 최소화
- 신뢰감 = 투명성 (단가 출처, 계산 근거를 숨기지 않음)

### 다크/라이트
- 헤더: 딥 브라운 배경 + 크림 텍스트
- 입력 패널: 크림 배경 (#F5F0EB)
- 결과 패널: 화이트 배경
- 배너 영역: 연한 크림 배경으로 본문과 자연스럽게 구분
- 푸터: 딥 브라운 배경

## 4. 입력 항목

### 4-1. 평형 (필수)
- 선택지: 10평 / 20평 / 24평 / 30평 / 34평 / 40평 / 50평 / 직접입력
- UI: 버튼 그룹 (카드형 또는 칩 형태)
- 기본값: 30평

### 4-2. 주거 유형 (필수)
- 선택지: 신축 입주 / 구축 리모델링(10년 미만) / 구축 리모델링(20년 이상)
- UI: 라디오 버튼 또는 세그먼트 컨트롤
- 기본값: 구축(20년 이상)
- 계수: 신축 0.7 / 구축10년 0.9 / 구축20년 1.0

### 4-3. 지역 (필수)
- 선택지: 서울 / 경기·인천 / 지방광역시 / 기타 지방
- UI: 드롭다운 또는 세그먼트
- 기본값: 서울
- 계수: 서울 1.0 / 경기 0.92 / 지방광역시 0.85 / 기타 0.78

### 4-4. 자재 등급 (필수)
- 선택지: 기본(가성비) / 중급(대중적) / 고급(프리미엄)
- UI: 3개 카드 (각 카드에 대표 자재/브랜드 예시 표시)
- 기본값: 중급
- 각 등급별 대표 예시:
  - 기본: 합지벽지, 장판 1.8T, 국산 세라믹, 사제 싱크대
  - 중급: 실크벽지, 강마루 6T, 포세린 타일, 엔지니어드스톤 싱크대
  - 고급: 수입벽지, 원목마루, 수입 포세린, 세라믹/브랜드 싱크대

### 4-5. 천장 높이 (선택)
- 선택지: 2,300mm(표준) / 2,400mm / 2,500mm 이상
- 기본값: 2,300mm
- 계수: 2300 = 1.0 / 2400 = 1.05 / 2500+ = 1.10

### 4-6. 공정 선택 (토글)
각 공정별 ON/OFF 토글. 기본값은 전부 ON.
일부 공정에는 세부 옵션 (개수 선택 등) 포함.

|------|-----------|--------|

| 전기 | - | ON |

## 5. 결과 출력 항목

### 5-1. 총 예상 비용 (가장 크고 눈에 띄게)
- 예: "약 4,255만원"
- 예비비(15%) 포함 금액
- 평당 단가 함께 표시: "평당 약 142만원"

### 5-2. 합리성 판단 배지
- 시장 평균 대비 위치 판단
- 색상으로 구분: 그린(합리적) / 앰버(적정) / 레드(다소 높음)
- 한 줄 코멘트: "30평 구축 리모델링 중급 기준, 경기 지역 평당 142만원은 시장 평균 범위 내 적정 수준입니다."

### 5-3. 공정별 견적표
- 테이블 형태: 공정명 / 금액 / 비중(%) / 수평 바
- 금액 큰 순서로 정렬 (어디에 돈이 가장 많이 가는지 직관적으로)

### 5-4. 비용 비중 차트
- 도넛 차트 또는 수평 바 차트
- "돈이 가장 많이 가는 곳" 시각화

### 5-5. 절약 팁 (선택적)
- "욕실 타일 등급을 국산 세라믹으로 낮추면 약 300만원 절약 가능"
- "시스템 에어컨 대신 벽걸이로 변경하면 약 280만원 절약 가능"

### 5-6. 공정 순서 + 예상 기간
- 타임라인 형태
- 어떤 공정이 먼저이고 전체 몇 주 걸리는지

### 5-7. 다운로드

**PDF 다운로드 (비회원 가능)**
- 공정별 견적표 + 차트 + 합리성 판단 + 공정 순서를 A4 리포트로 출력
- 개인정보 입력 없이 바로 다운로드
- 하단에 "2025~2026 시장 평균 기반 · 조달청 적산 교차 검증 예정" 표시

**엑셀 다운로드 (비회원 가능)**
- 시트1: 공정별 상세 견적표 (항목/수량/단가/소계, 수식 포함 — 사용자가 직접 수정 가능)
- 시트2: 비용 비중 요약 + 예비비 + 총 예산
- 시트3: 추천 업체 리스트 (지역별 협력 자재 매장·시공 업체 정보 — 광고 수익 지면)
- 시트4: 공정 순서 가이드 + 예상 기간
- 면책 문구 포함
- 추천 업체 시트는 "제휴" 표시 명확히

→ 엑셀 파일은 저장·공유될 때마다 추천 업체 리스트가 함께 확산되므로 자연스러운 B2B 광고 채널이 됨

### 5-8. 공유하기
- 카카오톡 공유 버튼
- 결과 카드 이미지: "30평 중급 리모델링 예상 견적: 약 4,255만원 — 얼마드나에서 확인"
- 링크 공유 시 OG 태그로 요약 정보 표시

## 6. 배너 광고 영역 + 엑셀 내 추천 업체

### 위치
- 결과 패널 하단 (공정별 견적표, 차트, 공유 버튼 아래)
- 콘텐츠 영역(견적표, 차트) 안에는 절대 삽입하지 않음

### 형태
- 협력 브랜드 배너 2~3개
- 가로형 배너 (728×90 또는 320×100 모바일)
- "광고" 또는 "제휴" 라벨 명시
- 팝업, 모달, 풀스크린 광고 절대 없음

### 자재 브랜드 노출
- 자재 등급 선택 시 각 등급 카드에 대표 브랜드명 표시
- 예: "중급 — 실크벽지 (예: LX Z:IN 베스띠)"
- 결과 견적표에서도: "바닥(강마루) — 동화자연마루 나투스강 기준"
- 이 노출은 광고가 아닌 "참고 정보" 형태. 별도 "제휴" 표시 불필요.

## 7. 회원 시스템 (선택적 가입)

### 핵심 원칙
- 비회원도 모든 기능 100% 사용 가능 (계산, PDF 다운, 엑셀 다운, 공유)
- 회원가입은 "내 견적 저장"을 원하는 사용자만 선택적으로
- 가입 절차는 최소화 — 카카오 로그인 원클릭 (이메일+비밀번호 가입 안 함)

### 비회원 vs 회원 비교

| 기능 | 비회원 | 회원 (카카오 로그인) |
|------|--------|---------------------|
| 견적 계산 | ✅ 무제한 | ✅ 무제한 |
| PDF 다운로드 | ✅ | ✅ |
| 엑셀 다운로드 | ✅ | ✅ |
| 카카오톡 공유 | ✅ | ✅ |
| 내 견적 저장 | ❌ | ✅ |
| 저장 견적 불러오기 | ❌ | ✅ |
| 여러 버전 비교 | ❌ | ✅ (최대 10건) |

### UI 처리
- 회원가입 유도 팝업/모달 절대 없음
- "저장하기" 버튼 클릭 시에만 "카카오로 로그인하면 저장할 수 있어요" 안내
- 로그인 후에도 화면 변화 최소화 (저장 버튼이 활성화되는 정도)

## 9. 단가 데이터 구조

### 데이터 소스
단가 DB는 별도 엑셀 파일로 정리 완료. (인테리어_자재_시장단가_DB.xlsx)
이 엑셀을 JSON으로 변환해서 프론트엔드에 내장.

### 계산 로직
```
공정별 금액 = 기본 단가[등급] × (평형 / 30) × 지역계수 × 주거유형계수 × 천장높이계수
총 소계 = Σ(ON 상태인 공정의 금액)
예비비 = 소계 × 0.15
총 예산 = 소계 + 예비비
평당 단가 = 총 예산 / 평형
```

### 계수표
| 구분 | 항목 | 계수 |

## 13. 확장 로드맵 — "인테리어의 부동산계산기가 된다"

부동산계산기(ezb.co.kr)가 중개수수료 계산기 하나로 시작해서 양도세·취득세·DTI·DSR·LTV 등 12개 이상 계산기를 갖춘 플랫폼이 된 것처럼, 얼마드나도 "인테리어 예상 견적 계산기" 하나로 시작해서 인테리어 관련 모든 계산·도구를 모은 플랫폼으로 확장한다.

### Phase 1 — 예상 견적 계산기 (현재)
평형·유형·지역·등급·공정 입력 → 예상 견적 산출 + 합리성 판단

### Phase 2 — 도구 확장
- **자재 소요량 계산기**: 평형 입력하면 벽지 몇 롤, 타일 몇 박스, 마루 몇 평 필요한지 자동 산출
- **견적서 비교 분석기**: 업체 견적서 항목별 금액 입력 → 시장 평균과 비교 ("이 항목은 시장 대비 30% 비쌉니다")
- **공사 기간 계산기**: 선택한 공정 기준 예상 공사 일수 + 공정 순서 간트차트
- **인테리어 대출 이자 계산기**: 공사비 대출 시 월 상환액 시뮬레이션

### Phase 3 — 콘텐츠 플랫폼화
- **자재 가격 비교**: 동일 자재의 쇼핑몰별 최저가 비교 (크롤링 또는 제휴)
- **시공 사례 DB**: 평형·스타일별 실제 시공 사례 + 실제 비용 공개
- **AI 견적서 분석**: 견적서 사진/PDF 업로드 → OCR + AI 자동 분석

핵심은 Phase 1을 최대한 빨리 런칭하고, 트래픽 반응을 보면서 Phase 2 도구를 하나씩 추가하는 것. 부동산계산기도 처음부터 12개를 만든 게 아니라 하나씩 붙여나간 거니까.

각 도구는 얼마드나 메인 페이지 상단 네비게이션에서 접근:
```
[견적 계산기] [자재 소요량] [견적서 비교] [공사 기간] [대출 이자]
```
